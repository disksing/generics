[![Go Report Card](https://goreportcard.com/badge/github.com/disksing/generics)](https://goreportcard.com/report/github.com/disksing/generics)


## Usage

Import package:

```
import "github.com/disksing/generics"
```

Enable generics:

```
generics.Enable()
```
